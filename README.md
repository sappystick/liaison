## 🚀 Quick Start

### Prerequisites
- Node.js 24 LTS
- Docker & Docker Compose
- 4+ CPU cores, 8GB RAM
- 100GB+ storage

### Installation
```bash
# Clone the repository
git clone https://github.com/sappystick/liaison.git
cd liaison

# Set up environment
chmod +x setup-liaison.sh
./setup-liaison.sh

# Start all services
docker-compose up -d

# Access the platform
open http://localhost:3000
```
