## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+
- Docker & Docker Compose
- 2GB RAM minimum

### Installation
```bash
# Clone the repository
git clone https://github.com/sappystick/liaison.git
cd liaison

# Set up environment
cp .env.example .env
# Edit .env with your API keys

# Start all services
docker-compose up -d

# Access the platform
open http://localhost:3000
```
