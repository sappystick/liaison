## 🎯 The Universally Beneficial AI Chatbot Platform

lIAison is a **containerized, no-code AI chatbot platform** designed specifically for **small businesses and solopreneurs**. Create, deploy, and monetize AI chatbots in minutes without any coding required.

## 🚀 Core Value Proposition

**Problem**: Small businesses need affordable, 24/7 customer engagement but lack budget for full-time support staff or custom development.

**Solution**: Plug-and-play AI chatbots with visual builder, instant deployment, and NFT-based template marketplace.

## 🏗️ Technical Architecture

### Simple Container Stack
```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   Frontend      │    │   Backend       │    │   Chatbot       │
│   Next.js       │◄──►│   Node.js       │◄──►│   Runtime       │
│   Visual Builder│    │   Express       │    │   AI Engine     │
└─────────────────┘    └─────────────────┘    └─────────────────┘
         │                       │                       │
         └───────────────────────┼───────────────────────┘
                                 │
         ┌─────────────────┐    ┌─────────────────┐
         │   Database      │    │   Blockchain    │
         │   PostgreSQL    │    │   NFT Contract  │
         │   Redis Cache   │    │   Ethereum      │
         └─────────────────┘    └─────────────────┘
```

### Tech Stack
- **Frontend**: Next.js 15 with React Flow for visual builder
- **Backend**: Node.js with Express and PostgreSQL
- **AI**: OpenAI GPT integration with custom prompts
- **Blockchain**: Simplified ERC-721 NFT contract
- **Deployment**: Docker Compose for single-command setup

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- Docker & Docker Compose
- 2GB RAM minimum

### Installation
```bash
# Clone the repository
git clone https://github.com/sappystick/liaison.git
cd liaison

# Set up environment
cp .env.example .env
# Edit .env with your API keys

# Start all services
docker-compose up -d

# Access the platform
open http://localhost:3000
```
